# CNCF Positioning Strategy

## Ecosystem Fit
USAGIX complements, not replaces:
- Kubernetes runtime interfaces (CRI/CNI/CSI)
- OCI runtime contracts
- OpenTelemetry observability standards

## Proposed Working Relationships
- TAG Security: threat model and isolation guidance
- TAG Runtime: substrate abstraction and runtime contracts
- OpenTelemetry SIG: semantic conventions alignment
- SPIFFE/SPIRE community: workload identity model alignment

## Deliverables for Community Review
- Reference implementation interoperability report
- Conformance test harness
- Versioned API evolution policy
