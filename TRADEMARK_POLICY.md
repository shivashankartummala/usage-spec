# Trademark Policy

## Registered Marks

The following marks are owned by the USAGE Specification Stewards:
- **USAGE** — The USAGE specification itself
- **USAGE-compliant** — May be used to denote compliant implementations (with permission)

## Trademark Usage Guidelines

### ✅ Permitted Uses (No Permission Required)

1. **Referential use**: "We implement the USAGE specification" or "USAGE-compliant substrate"
2. **Specification references**: "As specified in USAGE v1.0"
3. **Truthful descriptions**: "Based on USAGE" (if genuinely compliant)

### ❌ Prohibited Uses

1. **Confusion**: "USAGE" as a company or product name (suggests false affiliation)
2. **Modification**: "USAGE-Plus," "USAGE-Enterprise," "USAGE Pro" (unless separately branded)
3. **Domain registration**: Do NOT register usage.io, usage.app, etc. without permission
4. **Implied endorsement**: Using the mark to suggest endorsement by USAGE Stewards (unless explicit)

### ⚠️ Requires Permission

- Logo usage
- Certification marks (e.g., "USAGE-Certified")
- Use in domain names or product names
- Any use suggesting official affiliation

**To request permission**: Email trademark@usage-spec.io with description of intended use.

## Enforcement

Trademark violations will be addressed through:
1. Informal notice and discussion
2. Formal takedown letter (if informal fails)
3. Legal action (if necessary)
