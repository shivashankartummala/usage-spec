# Trademark Policy

## Registered Marks

The following marks are owned by the USAGIX Specification Stewards:
- **USAGIX** — The USAGIX specification itself
- **USAGIX-compliant** — May be used to denote compliant implementations (with permission)

## Trademark Usage Guidelines

### ✅ Permitted Uses (No Permission Required)

1. **Referential use**: "We implement the USAGIX specification" or "USAGIX-compliant substrate"
2. **Specification references**: "As specified in USAGIX v1.0"
3. **Truthful descriptions**: "Based on USAGIX" (if genuinely compliant)

### ❌ Prohibited Uses

1. **Confusion**: "USAGIX" as a company or product name (suggests false affiliation)
2. **Modification**: "USAGIX-Plus," "USAGIX-Enterprise," "USAGIX Pro" (unless separately branded)
3. **Domain registration**: Do NOT register usage.io, usage.app, etc. without permission
4. **Implied endorsement**: Using the mark to suggest endorsement by USAGIX Stewards (unless explicit)

### ⚠️ Requires Permission

- Logo usage
- Certification marks (e.g., "USAGIX-Certified")
- Use in domain names or product names
- Any use suggesting official affiliation

**To request permission**: Email trademark@usage-spec.io with description of intended use.

## Enforcement

Trademark violations will be addressed through:
1. Informal notice and discussion
2. Formal takedown letter (if informal fails)
3. Legal action (if necessary)
